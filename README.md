## End-to-End Customer Analytics: Churn & NLP

A hands-on data science project that analyzes e-commerce customer data to predict churn and extract complaint insights using NLP.

## Overview

This project covers three main areas:
- **Data Exploration** - Cleaning data, handling missing values, and creating visualizations to understand churn patterns
- **Predictive Modeling** - Building a Random Forest classifier to predict customer churn
- **Text Analysis** - Using TF-IDF to extract key complaint keywords from churned customer reviews

## Results

| Metric | Value |
|--------|-------|
| Model Accuracy | 71.2% |
| Top Churn Predictor | Yearly Amount Spent |
| Top Complaint Keywords | customer service, quality, slow delivery |

## Tech Stack

- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (Random Forest)
- TF-IDF (sklearn), WordCloud
